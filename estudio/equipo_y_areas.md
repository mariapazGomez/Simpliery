# 3 · Equipo y áreas para escalar

> **Leyenda:** 🟡 supuesto de mercado · 🔮 proyección. Sueldos en **CLP brutos/mes** (rangos de mercado chileno 2026, 🟡 estimados; validar con Get on Board / Talently / Glassdoor CL antes de contratar). 1 USD = CLP 905.
> ARR en USD y su equivalente anual CLP: **US$250K ≈ $226M · US$1M ≈ $905M · US$3M ≈ $2.715M · US$6M ≈ $5.430M.**

---

## 3.1 Las 5 áreas funcionales de un SaaS

| Área | Qué resuelve | En Control Local, prioridad |
|---|---|---|
| **Producto / Ingeniería** | Construir y mantener la app, cerrar el foso (boleta SII), confiabilidad | 🔴 Crítica desde el día 1 |
| **Ventas** | Convertir interesados en clientes pagando (autoservicio + asistido) | 🔴 Crítica para escalar |
| **Marketing / Growth** | Generar demanda barata (clave con ARPU bajo: el CAC manda) | 🔴 Crítica |
| **Customer Success / Soporte** | Onboarding, retención, bajar churn (a este precio, **la retención es la rentabilidad**) | 🟠 Crítica a partir de US$250K |
| **Finanzas / Ops** | Cobranza/suscripción, cumplimiento, datos, contabilidad propia | 🟡 Ligera al inicio, formal desde US$1M |

**Principio rector dado el ARPU bajo (US$11–39/mes):** no se puede pagar un equipo de ventas caro por cliente. El modelo **debe ser product-led** (autoservicio, onboarding sin humano, viralidad) con ventas asistidas solo para el plan Pro/cuentas con despacho. El equipo se diseña alrededor de eso.

## 3.2 Plan de contratación por fases (ligado a hitos de ingreso)

### Fase 0 — Pre-tracción / bootstrap (hoy → ~US$50K ARR)
**Objetivo:** cerrar boleta SII, cobrar a los primeros 100–400 negocios, validar retención.

| Rol | Cuántos | Sueldo CLP/mes 🟡 | Nota |
|---|---|---|---|
| Fundador (producto + dev + ventas) | 1 | — (equity / sueldo mínimo de subsistencia) | Hace de todo |
| Dev full-stack (contrato/part-time o cofundador técnico) | 0–1 | $2,5M–3,5M (o equity) | Acelera SII + estabilidad |
| Soporte/onboarding (part-time o freelance) | 0–1 | $600k–900k | Puede ser el fundador al inicio |

> **Costo mensual aprox.:** $0–4M. **Foco:** producto y los primeros clientes pagando. Nada de estructura todavía.

```
        Fundador/a
        (todo)
            │
   ┌────────┴────────┐
 Dev (1, opt.)   Soporte (opt.)
```

### Fase 1 — Tracción inicial (~US$250K ARR ≈ 1.000–1.200 clientes)
**Objetivo:** motor de adquisición repetible + retención sana. Sale el fundador del soporte 1-a-1.

| Rol | Cuántos | Sueldo CLP/mes 🟡 |
|---|---|---|
| Fundador → CEO/Producto | 1 | — |
| Dev full-stack (semi-senior/senior) | 1–2 | $2,5M–4,0M |
| Customer Success / Soporte | 1 | $1,0M–1,5M |
| Growth/Marketing (contenido + ads + Sercotec) | 1 | $1,5M–2,5M |
| Contador externo (servicio) | — | $400k–800k/mes |

> **Equipo: ~4–5 personas.** Costo nómina aprox.: **$7M–13M/mes (~US$90K–170K/año)** — sostenible dentro de US$250K ARR si el crecimiento es real.

```
                 CEO / Producto
        ┌──────────────┼──────────────┐
   Ingeniería       Growth        Customer Success
   (1–2 devs)     (1 marketing)     (1 CS/soporte)
                                  Contador externo ─┘
```

### Fase 2 — Escalamiento (~US$1M ARR ≈ 4.000–5.000 clientes)
**Objetivo:** especializar funciones, profesionalizar ventas asistidas (plan Pro/despacho), datos.

| Rol | Cuántos | Sueldo CLP/mes 🟡 |
|---|---|---|
| CEO | 1 | $3,0M–5,0M |
| Líder técnico / CTO | 1 | $4,0M–6,0M |
| Devs (full-stack + 1 móvil) | 3–4 | $2,5M–4,5M |
| Diseñador producto/UX | 1 | $2,0M–3,0M |
| Head de Growth/Marketing | 1 | $2,5M–4,0M |
| Ejecutivo de ventas (Pro/despacho) | 1–2 | $700k base + comisión (~$1,5M OTE) |
| Customer Success | 2–3 | $1,0M–1,8M |
| Ops/Finanzas (interno) | 1 | $1,8M–2,8M |

> **Equipo: ~11–14 personas.** Costo nómina aprox.: **$28M–42M/mes (~US$370K–560K/año)** — ~40–55% del ARR, rango típico de SaaS en crecimiento.

```
                         CEO
        ┌────────────┬───────────┬────────────┬───────────┐
       CTO        Growth      Ventas        CS Lead     Ops/Finanzas
   3–4 devs +   1 Head +    1–2 ejec.     2–3 CS         (1)
   1 diseñador  contenido
```

### Fase 3 — Rumbo a US$3M → US$6M ARR (≈ 13.000–30.000 clientes y/o expansión regional)
**Objetivo:** máquina de adquisición a escala, posible expansión Perú/Colombia, fintech.

| Área | Headcount aprox. 🔮 |
|---|---|
| Ingeniería + Producto + Datos | 12–18 |
| Growth/Marketing | 4–6 |
| Ventas | 4–6 |
| Customer Success/Soporte | 8–12 |
| Finanzas/Ops/Legal | 3–5 |
| Liderazgo (C-level/heads) | 4–5 |

> **Equipo: ~35–50 personas** a US$3M; **~50–70** rumbo a US$6M. Costo nómina objetivo: **≤ 45–55% del ARR.** A esta altura típicamente entra capital de riesgo (Serie A) para financiar la adquisición por delante del ingreso.

## 3.4 Notas y advertencias

- **No contratar adelantado al ingreso.** Con ARPU bajo, cada sueldo "antes de tiempo" quema caja rápido. Regla: una contratación nueva debe pagarse sola con el ingreso incremental que habilita en ≤ 6–9 meses.
- **El primer hire crítico es técnico** (cofundador/CTO) para cerrar **boleta SII** y la **deuda de multi-tenant de OptiRoute** — sin eso no hay producto vendible al segmento que paga.
- **El segundo es Customer Success**, no un vendedor. A este precio, **el crecimiento viene de retener + referir**, no de vender uno a uno.
- **Sercotec / canales públicos** ("Digitaliza tu Almacén") pueden hacer las veces de fuerza de ventas subsidiada en la Fase 1–2 → reduce la necesidad de headcount comercial. Ver [plan de escalamiento](plan_escalamiento.md).
- 🟡 Todos los sueldos son rangos de mercado estimados; **validar** con fuentes salariales chilenas actualizadas (Get on Board, Talently, Glassdoor Chile) al momento de contratar.
